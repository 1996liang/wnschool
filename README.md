# wnschool
一个跑腿微信小程序前端部分，嵌入了微信支付，订单管理，发单接单，论坛社区，点赞，钱包等等功能<br>
由于后台服务器已经过期，效果图有些采取原本的静态设计页面，有的采取获取不到数据后的页面。但作为一个微信小程序，它的第一版本是完整的。<br>
